Kitter

Welcome to kitter's page! Kitter is a discord bot for Black Cat Studios' Discord Server. Kitter has a ton of commands and features for you to check out.

Commands

List of all commands. Arguments in "[ ]" are required, "< >" means they are not.
Commands marked as context mean they are only avaliable through the context menu. (ex: Right click > Apps > Command)
Context commands with the MESSAGE type are avaliable by right clicking a message in the context menu, same with USER
Commands with the SLASH type can be accessed by typing: "/[command]" Subcommands can be accessed by typing: "/[basecommand] [subcommand]"

Features

Ranking system

Kitter gives you points based on your activity. Points raise your level, which in turn give you roles. The following actions reward points:

Sending messages

Being in voice chat

Inviting members

Interacting with the bot

Reception system

Kitter has autorole and manages new users. You must hit a green button to verify, this lowers the chances of raids. Additionally, kitter manages ping selections so that you can control which announcements you'd like to be pinged for.

Announcement system

Kitter manages announcements, showing them in a pretty embed. Kitter can also post announcements here to the website allowing for us to easily add to the newsfeed without needing to know how to code a bit.

Audit log

Kitter shows an in-depth audit log of all events that take place. Displaying reasons, and even tracking invite uses to network possible threats.
